Managing images on server

  • /uploads: lists files on server
  • /uploads/'filename' : returns the file specified from the server
  • /upload: upload a file
  • /clearuploads clears the uploaded files folder.

Use one of the following methods and the arguments listed

  • thresholding method
    • filename: *required* the file to process
    • templateSize: the size of template to use
  • templating method
    • filename: *required* the file to process
    • threshold: the threshold value for example 220
